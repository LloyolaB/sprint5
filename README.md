
# Sprint Modulo 5

Sistema de registros de usuarios y capacitaciones para un servicio de Asesorias en Prevencion de riesgo.


## Caracteristicas

- Gestion de usuarios (Administrativo, Profesional, Cliente)
- Gestion de Capacitacion
- Hecho en Bootstrap
- Java EE 8
